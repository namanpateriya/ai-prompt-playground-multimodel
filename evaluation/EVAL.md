# Evaluation

Evaluate the outputs generated by the Multi-Model Prompt Playground.

---

## Setup

Ensure API keys are configured in `.env`.

---

## How to Run Evaluator

```bash
python -m evaluation.evaluator
```

---

## How to Run Optimizer

```bash
python -m evaluation.optimizer
```

---

## Input Required

Test cases are defined in:

evaluation/test_cases.json

Each test case includes:

* input prompt

---

## Sample Output

Input: Explain artificial intelligence

{
"concise": {

}
}

---

## Evaluation Summary

The evaluator checks:

* number of variations generated
* presence of provider errors

---

## Notes

* This evaluation focuses on execution reliability, not semantic correctness
* Extend to compare outputs across models for deeper analysis
